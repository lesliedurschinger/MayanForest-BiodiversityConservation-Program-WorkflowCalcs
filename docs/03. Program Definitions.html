Project Extent

The delineation of the Project Extent followed a stepwise geospatial approach. A shapefile representing the complete areas under ejido control was used, reflecting lands over which participating communities hold legal ownership and management rights. This dataset served as the foundational boundary for the program. The ejido shapefile was then spatially refined by clipping it to areas classified within the IUCN Global Ecosystem Database, ensuring alignment with recognized ecosystem typologies used in the methodology. The resulting intersected area represents the final Project Extent, within which all project activities are implemented, and biodiversity outcomes are measured. The ecosystem typology for the Project Extent was determined differently than the instructions of the Nature Framework. This is a project deviation that was needed to accurately define the ecosystem typology in the Project Extent. When using the IUCN GET level 3 classifications and tool for the Project Extent (part of the Yucatan Peninsula, Mexico), the data was inaccurate and deficient. For instance, the main function groups that were found in the Project Extent included intensive landscapes like annual croplands, plantations, and urban and industrial ecosystems, as seen in Table 2.

However, these functional groups are not true for the Project Extent, as they are forested areas. Terra Global has worked closely with Amigos de Calakmul, A.C. (ACAC), implementing partner in this area for over a decade, and recently conducted ground truthing for a carbon project in the same area where extensive LU-LC was assessed. These areas are primarily forested and many of the ejidos included in the program are in the buffer zone of the Calakmul Biosphere Reserve, a largely intact Mayan forest remnant home to much of the wildlife and one of the largest populations of jaguars left in the Yucatan peninsula. Thus, the data used to define the ecosystem types of the Project Extent was based on research and literature (Jung et al. 2020). The data was based on the IUCN Global Database that was later created and used for the methodology, so it is still appropriate to use in this case. This data also allowed the program to refine the Project Extent to the nine ejidos that are included in the Program, and it provided much more accurate results.

The main functional group that was found in all of the ejidos was the Tropical/Sub-tropical Dry forests and thickets. While there were several ecosystem types found in each ejido (Map 1), these ecosystem types were excluded due to being arable land, plantations, or urban areas that are not eligible under the methodology. Furthermore, other ecosystem types, like wetlands and sub-tropical/tropical swamps only made up 1.44% and 0.78% respectively, in the largest areas of the ejidos. Project Extent Calculations

While dry savanna make up part of the ecosystem types for some ejidos, it was ultimately excluded from the Project Extent, as program activities are centered to be implemented in the dry forest ecosystems, and any ecosystem that makes up less than 35% of an ejido is excluded due to their limited spatial extent, which reduces their influence on landscape-scale ecological processes and biodiversity outcomes. Prioritizing ecosystem types that represent a more substantial portion of the Project Extent allows for more efficient allocation of resources, more robust monitoring, and greater potential for measurable impact. However, dry savannas are native ecosystems in the project extent and future instances will include this ecosystem.
